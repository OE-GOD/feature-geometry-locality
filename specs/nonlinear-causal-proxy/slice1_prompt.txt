Mechanism:
1. Load SAE weights W_dec (24576,768), W_enc (768,24576), b_enc (24576,), b_dec (768,) from the same sae_weights.safetensors run_layer.fetch uses (extend the loader). Load GPT2LMHeadModel + tokenizer (as causal_proxy.py does). Build a small diverse corpus of token windows (reuse/extend causal_proxy.build_contexts; ~48 windows, seqlen 32).
2. SAE encoder for a residual vector h (d_model): a = relu((h - b_dec) @ W_enc + b_enc)  -> (24576,) feature activations. Reconstruction: recon = a @ W_dec + b_dec.
3. Register a forward_pre_hook on model.transformer.h[LAYER] (its input == blocks.LAYER.hook_resid_pre). Baseline pass (no ablation): record clean logits AND the per-position residual h for every (context,position); from h compute feature activations a for the K target features.
4. Target features: alive features (log_sparsity > DEAD_LOG_SPARSITY), subsample K=200 with numpy default_rng(seed). A feature "fires" at a position if a_i > 0. Keep per-feature list of firing positions.
5. For each target feature i: one forward pass where the hook SUBTRACTS a_i(pos)*W_dec[i] from h at exactly that feature's firing positions (leave other positions unchanged); collect ablated logits. effect at a firing position = clean_logits[pos] - ablated_logits[pos] (the vocab-length next-token logit vector at that position). F_i = mean of that over the feature's firing positions. Reduce to the (vocab,) mean vector in float32 BEFORE any float64 cast (avoid materializing doubled (batch,seq,vocab) tensors — this was a real 4x slowdown before).
6. Features that fire at fewer than MIN_FIRES (default 5) positions: set F_i to zeros and record the low count (don't crash).
